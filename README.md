# Network Scanner

A powerful and user-friendly network scanner built with Next.js and TypeScript.

## Features

- Device Discovery
- Port Scanning
- Network Information Analysis
- Fast and Responsive UI
- Built with Next.js, TypeScript, and Tailwind CSS

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Node.js

## Installation

```bash
git clone https://github.com/AbhishekTor55/Network-Scanner.git
cd network-scanner
npm install
npm run dev
